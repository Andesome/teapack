## teapack

Just always be fresh.
永远保持最新技术支持的一个自定义、自由(0配置)脚手架。


## 命令行参数
- -d,--debug  开启调试模式

1. 启动失败，控制台输出详细日志
2. 打包失败，控制台输出详细日志

- -dev, --development 开发模式

1. 即webpack development模式
2. 开启source map，调试代码


- -prod, --production 生产模式

1. 即webpack production模式
